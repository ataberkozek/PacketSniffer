---Packet Sniffer---

How to run:

1) Run pycharm with SUDO permission (without root permission the program is not going to run as expected).

2) Run packet_sniffer.py.

3) Open up a browser.

4) Press initialize button.

5) Reload a page on the browser. 

- Additionally some images that shows the structure of the Ethernet Frame, Ip header
and TCP IP Packet is added in the "Basics" folder of the project.
